## Build Blog CMS (Content Management System) with Laravel 7

## Clone this repo
```
https://github.com/imbartek/blog.git
```

## Install composer packages
```
$ cd blog
$ composer install
$ npm install
$ npm run dev
```

## Deploy on localhost

$ php artisan serve


## Mail setup 
```
visit at : https://mailtrap.io/
put mail credentials in .env file
```
